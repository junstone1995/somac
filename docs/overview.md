# Somac — Project Overview

## 앱 개요

맥주잔의 실제 크기를 AR로 측정하여 황금 비율의 소맥을 만들 수 있도록 돕는 iOS 앱.
잔마다 크기가 달라도 정확한 비율로 따를 수 있게 폰을 세워 시각적 가이드를 제공한다.

---

## 핵심 기능 (v1 — 소맥)

### 1. 잔 측정 (AR Measurement)
- Apple ARKit + LiDAR Scanner 기반
- **맥주잔만 측정** (소주잔 측정 불필요 — 소주 25ml 고정)
- 측정 항목 (맥주잔)
  - **바닥 둘레** (bottom circumference)
  - **상단 둘레** (top circumference)
  - **높이** (height) — 바닥·상단 평면 간 거리로 자동 계산
- 측정 흐름: 바닥 테두리 포인트 → 상단 테두리 포인트 → 자동 완료
- 계산: 원뿔대(frustum) 부피 — 테이퍼드 잔 정확 대응
  - `r1 = bottom circumference / (2π)`
  - `r2 = top circumference / (2π)`
  - `volume = (π × h / 3) × (r1² + r1×r2 + r2²)`

### 2. 황금 비율 계산 (확정)
- **소주: 25ml 고정** (반잔, 별도 측정 없음)
- **맥주: 50ml** (소주의 2배)
- 소주 : 맥주 = 1 : 2
- 맥주잔 부피 기반으로 맥주의 **채울 높이(fill height)** 역산

### 3. 세로 모드 시각적 가이드
- 폰을 세웠을 때 (portrait) 화면 활성화
- 화면을 물리적 자(ruler)로 사용 — 460 PPI 기준으로 화면 높이를 실제 mm로 환산
- **폰을 잔 옆에 세우면** 화면의 선 위치 = 잔의 실제 액체 높이와 1:1 물리적으로 일치
- AR로 측정한 잔의 높이(mm)와 채울 높이(mm)를 화면 좌표로 변환
  - 잔이 화면보다 짧으면 1:1 물리 매핑 (선 위치 = 실제 mm 높이)
  - 잔이 화면보다 길면 화면에 맞게 비례 축소
- **파란 선**: 소주 25ml 채울 높이 — 바닥에서부터 이 선까지 소주를 따름
- **노란 선**: 소주 25ml + 맥주 50ml = 75ml 채울 높이 — 파란 선 위로 이 선까지 맥주를 따름
- 실시간으로 잔에 따르는 것처럼 채워지는 애니메이션
